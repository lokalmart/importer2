Standar sheet universal

Nama sheet = nama model Odoo, contoh: project.task

Kolom khusus:
_action       upsert / create / update / delete / skip
_external_id  external id stabil, contoh lokalmart_importer.page_scan
_match        field pencari, contoh name,project_id
_model        opsional bila sheet bukan nama model

Relasi:
project_id       nama/id/xmlid otomatis
project_id/id    paksa ID
project_id/name  paksa nama
tag_ids          many2many dipisah koma

Lokalmart Universal Odoo XLSX Importer V6.1

Satu importer untuk pages, custom fields, project, produk, course, dan data Odoo Lokalmart.

Format XLSX dibuat universal: nama sheet adalah nama model Odoo, atau isi kolom _model. Importer akan membaca fields asli dari Odoo, memvalidasi kolom, resolve relasi many2one/many2many, lalu create/update otomatis.

Standar sheet universal

Nama sheet sebaiknya langsung nama model, misalnya project.task. Kolom khusus: 

_model        opsional jika nama sheet bukan nama model
_action       upsert / create / update / delete / skip
_external_id  external id stabil, contoh lokalmart_importer.scan_page
_match        field pencari, contoh name,project_id

Contoh relasi:
project_id       boleh angka ID, nama, atau external id
project_id/id    paksa sebagai ID Odoo
project_id/name  paksa cari berdasarkan nama
project_id/xmlid paksa cari external id
tag_ids          many2many, pisahkan dengan koma

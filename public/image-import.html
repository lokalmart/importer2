Lokalmart Image Import V10

Import foto produk langsung dari XLSX ke Odoo.

Gunakan kolom image_url, image_base64, atau image_filename. Untuk foto lokal, pilih file gambar lalu XLSX cukup menulis nama filenya.

Command Center

Koneksi Odoo

Image dikirim sebagai base64 ke field seperti image_1920. API key lebih aman daripada password.

File XLSX & Foto Lokal

Embedded image yang ditempel di Excel belum otomatis dibaca. Pakai image_url, image_filename, atau image_base64.

Format Kolom XLSX

model              opsional, default product.template
default_code       SKU produk
barcode            barcode produk
name               nama produk/partner
match_field        opsional
match_value        opsional
target_field       default image_1920
image_url          URL publik gambar
image_filename     nama file lokal yang dipilih
image_base64       base64 langsung
